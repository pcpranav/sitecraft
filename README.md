# Sitecraft v2 — AI Website Builder

An AI-powered website builder built with Claude. Describe any idea and get a complete multi-page website with HTML, CSS, and JS — ready to iterate, export, or deploy.

## Features
- Multi-page generation with working navigation
- Separate `shared.css` and `shared.js` per generated site
- AI-powered page iteration and refinement
- Add new pages on the fly
- ZIP download or single-page HTML export
- One-click Netlify deploy

## Deployment
This site is configured for Netlify. Connect this repo in Netlify and every push auto-deploys.

## Local Development
Just open `index.html` in a browser — no build step needed.
